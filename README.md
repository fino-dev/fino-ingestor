# fino_core

### about fino_core

fino_core では FINO PROJECT 全体でコアとなる機能群をパッケージングしたものとなります。
fino を利用してデータ分析を行う際に、ユーザの環境に関わらず再利用可能な機能を提供することを目的としてます。
そのため、ある程度の抽象化を行い、ストレージや DB などの具体的な環境にも port として提供したり、データ収集から、分析までのプロセスごとのユースケースに応じて柔軟に取り込めるように設計しております

## about FINO PROJECT

FINO PROJECT は、Python をベースとした OSS（オープンソースソフトウェア）として、
**誰もが個人投資家として、金融のスペシャリストと同じ視点で市場や企業を分析し、投資判断ができるようにすること**を目的としたプロジェクトです。
金融分野の分析に入門すると、**データの規模の大きさや、金融業界特有のドメイン知識の壁に早い段階で直面することが少なくありません**。
FINO はそうした障壁をできる限り吸収し、実践的な自動化や仕組み化するための基盤となる機能を提供することを目的としています。

金融データ分析に興味を持ち始めた方や、投資を始めたものの「より自分なりの評価基準を持ちたい」と感じている方でプログラミングに抵抗がない方を主な対象としたパッケージを目指しています。

### FINO's Other Repository

#### Fino Kit

FINO PROJECT の実践的なテンプレートとして、データ管理から分析までのプラットフォームを構築するためのレポジトリです

###### [Features]

- AWS を使用した、cloud 環境のセットアップ用 CDK template（メンテナーの好みにより typescript で実装）
- Grafana によるデータ可視化ダッシュボード
- Jupyter Notebook による分析実行環境とユーティリティ

#### fino_cli

fino_core をベースとしたデータ管理をローカル環境で構築・運用するための CLI Tool

###### [Features]

- ユーザ独自のデータ基盤をローカル環境で構築する際に、データ収集や、ウェアハウス（レイクハウス）の ETL 処理などの分析するために必要なプロセスを自動化して実行するためのクライアントツール
- Fino kit と統合して利用することを前提として構築されているが、さまざまなユースケースやユーザの環境に fit できるようなツールを目指す
